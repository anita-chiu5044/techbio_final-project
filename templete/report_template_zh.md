# 中文報告模板

格式狀態：參照本院/台大臨床報告習慣整理，待領域同仁核准。除非另有核准來源，不宣稱為台大醫院官方格式。

## 建議報告架構

```text
一、檢體資訊
- 檢體種類：
- 採檢日期/時間：
- 收件日期/時間：
- 檢查項目：

二、臨床資料
- 臨床診斷/疑問：
- 相關病史：
- 近期治療或檢查：

三、檢查結果
- 周邊血球形態：
- 骨髓抹片/切片所見：
- 特殊染色/流式細胞儀/分子或細胞遺傳學結果：

四、綜合判讀
- 主要形態學發現：
- 與臨床問題之相關性：
- 需排除或建議進一步確認事項：

五、結論/意見
- 結論：
- 建議：
- 是否需緊急複核或通知：
```

## 可填入的正式模板

```text
請在此填入領域同仁核准後的去識別化中文報告模板。
```

## 撰寫原則

- 不填入病人姓名、病歷號、身分證字號、生日、地址、電話或其他可識別資訊。
- 不直接複製完整 SOP 或原始報告影本。
- 結論應與形態學、CBC、流式細胞儀、分子或細胞遺傳學等資料一致。
- 若證據不足，使用「建議綜合臨床及其他檢驗結果評估」等保守語氣。
- 任何疑似緊急或高風險情境，應依 `review_triggers_zh.yaml` 與 `critical_flags_zh.yaml` 升級處理。
