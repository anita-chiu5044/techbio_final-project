# Reporting Guidelines Workspace

This folder is the fixed place for domain-teammate inputs that Qwen should follow when writing reports or answering case-level questions.

Put only structured, project-approved guidance here. Do not drop raw full SOP PDFs directly into prompts.

Recommended files:

```text
report_template.md
allowed_phrases.md
prohibited_claims.md
review_triggers.yaml
critical_flags.yaml
cell_abbreviation_canonical_map.md
qc_review_template.md
source_notes.md
```

What domain teammates should contribute:

```text
- de-identified report wording or template
- allowed wording for morphology suggestions
- prohibited wording for over-claiming diagnosis
- urgent review triggers, e.g. possible APL wording
- QC review checklist wording
- canonical cell abbreviation map
- notes from local hospital / lab SOP discussion
```

Suggested rule:

```text
If a document is long or sensitive, summarize it into one of the structured files above and keep the raw source elsewhere.
```
