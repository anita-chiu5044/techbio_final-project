# 中文 QC 複核模板

本檔案用於定義報告品質複核時的標準檢核文字。

## 複核清單

- [ ] 指引內容未包含病人可識別資訊。
- [ ] 報告架構符合已核准模板。
- [ ] 形態學描述使用已核准用語。
- [ ] 未出現禁止的診斷過度宣稱。
- [ ] 已檢查 `review_triggers_zh.yaml` 中的緊急複核條件。
- [ ] 已檢查 `critical_flags_zh.yaml` 中的重大旗標。
- [ ] 細胞縮寫符合 `cell_abbreviation_canonical_map_zh.md`。
- [ ] 建議或限制說明未超出資料可支持範圍。
- [ ] 來源筆記已摘要化，且可追溯到核准來源。
