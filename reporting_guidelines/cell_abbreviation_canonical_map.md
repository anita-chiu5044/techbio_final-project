# Cell Abbreviation Canonical Map

Use this file to define canonical cell abbreviations and accepted synonyms.

| Canonical abbreviation | Full term | Accepted synonyms | Notes |
| --- | --- | --- | --- |
| NGS | Neutrophil, segmented | Seg, Segmented neutrophil, PMN | Mature granulocyte; most abundant WBC in normal blood |
| NGB | Neutrophil, band | Band, Band neutrophil, Stab | Immature granulocyte; elevated in left shift |
| LYT | Lymphocyte (typical) | Lymph, Small lymphocyte | Normal mature lymphocyte |
| LYA | Lymphocyte (atypical) | Atypical lymphocyte, Reactive lymphocyte, ALL blast | Includes reactive forms and ALL lymphoblasts in training data |
| MON | Monocyte | Mono | Largest normal WBC; elevated in M4/M5 |
| EOS | Eosinophil | Eo | Bilobed nucleus, orange-red granules |
| BAS | Basophil | Baso | Rare; elevated may suggest CML |
| EBO | Erythroblast | NRBC, Nucleated RBC, Normoblast | Erythroid precursor; elevated in M6 |
| MYO | Myelocyte | Myelo | Granulocyte precursor with specific granules |
| MMZ | Metamyelocyte | Meta | Between myelocyte and band; kidney-shaped nucleus |
| MYB | Myeloblast | Blast (myeloid) | Immature; dominant in M1/M2; clinically critical |
| PMO | Promyelocyte | Promonocyte | Granulocyte precursor; large azurophilic granules |
| PMB | Promonocyte | Promyelocyte (monocytic) | APL-relevant (M3); Auer rods; URGENT if dominant |
| MOB | Monoblast | | Monocytic blast; dominant in M5a; clinically critical |
| KSC | Smudge cell | Basket cell, Korbschatten | Artifact from fragile CLL lymphocytes; not a true cell type |

## Rules

- Add only abbreviations approved by the domain team.
- Prefer one canonical abbreviation per cell type.
- Include local naming differences in `Accepted synonyms`.
